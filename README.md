# react-native-webview-kit

support chart lib use webview on react native

## Installation

```sh
npm install react-native-webview-kit
```

## Usage

```js
import { WebviewKitView } from "react-native-webview-kit";

// ...

<WebviewKitView color="tomato" />
```
